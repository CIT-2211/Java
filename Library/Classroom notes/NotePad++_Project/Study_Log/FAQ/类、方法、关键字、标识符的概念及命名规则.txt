类、方法、对象、关键字、标识符的概念及命名规则

标识符：程序员自己命名的用于表示变量、方法、类等。
例如： int age;	 String Name;  double $salary;
中的age、Name、$alary
命名规则：
1、由字母、数字、_、$组成，但不能以数字开头。
2、区分大小写。
3、不能是Java的关键字。

关键字：Java语言中保留特殊意义的单词（就是Java自带的单词库）具有特殊含义，不可作为表示符。
例如: public、private、int、if、class、new、try等。
不能做变量和方法名

对象：通过new来创建，储存实际数据。每个对象独立储存自己的字段。（没对象就自己new一个）
例如：  Person giry = new Person();   giry是Person类的一个对象
	giry.name = "戴玛";           设置对象属性
	giry.SayLove();		      调用对象方法
对象是类的实例，对象和类的关系是多对一。

方法：类中定义一段可执行代码，用于描述对象的行为，可以自带参数和返回值。
分类：普通实例方法、静态方法、构造方法、带返回值的方法。
例如：  void SayLove(){System.out.println("I love you!");}普通实例方法,通过创建对象调用。
	static void issue () {……}	静态方法，通过调用了<类名>.<方法名>();调用。
	Persion() {……}		方法名与类名相同，无返回值。
	int SalaryAdd(int a){return10;}	声明返回类型，并用return返回类型。
Java中方法一定在类里面。

类：对象的蓝图（模板），描述对象的共同属性和行为。
组成：	字段（成员变量）:描述对象属性。（int age;String Name;）
	方法: 定义对象的行为。(void SayLove(){System.out.println("I love you!");})
	构造方法: 初始化对象。(Persion(){……})

Java文件的命名规则：
文件命名比标识符相比很宽松，除了不能以数字开头、不能包含类名、 \、 ?、 * 等非法字符，符合操作系统的文件命名规则就可以命名，但如public.java等以关键词命名的极度不推荐，编译或团队合作时候容易翻车。
